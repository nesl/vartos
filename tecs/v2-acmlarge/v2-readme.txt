ACM Transactions and Journals (Large Size) - V2 (April 2012)
------------------------------------------------------------

The following files are available in the v2-acmlarge.zip archive:

acmlarge.cls	     			- This is V1.3 of the LaTeX2e class file for the 'acmlarge template/format'
ACM-Reference-Format-Journals.bst	- This is the bibliography style file for the New ACM Reference Format (March 2012)
acmlarge-sample-bibfile.bib		- This is the bibliography database file
v2-acmlarge-guide.pdf			- This is a PDF of the "author guidelines' for the acmlarge template
v2-acmlarge-sample.pdf			- This is a PDF file showing what YOU should obtain
					  when YOU compile the sample source .tex file

v2-acmlarge-sample.tex			- the revised (v2) source sample .tex file 
v2-acmlarge-sample.bbl			- the bbl file as a result of 'BibTeX'ing 
acmlarge-mouse.eps			- Graphics file used in sample
acmlarge-mouse.pdf			- a graphics file in PDF format, (compatible with pdflatex)
algorithm2e.sty				- Algorithm package used in sample
url.sty					- URL package used in formatting the references
readme.txt				- This file!

Happy (La)TeXing!!!
